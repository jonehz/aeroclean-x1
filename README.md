# AeroClean X1

## Tech Stack
- Next.js 16
- TypeScript
- TailwindCSS v4
- React Hook Form
- Zod
- Framer Motion
- next-themes

## Features
- Responsive design
- Dark mode
- Form validation
- SEO optimization
- Accessibility support

## Run locally

npm install
npm run dev

## Build

npm run build
